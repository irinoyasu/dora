# Rails 7 Example App with Devise and Bootstrap

## Getting Started

### Requirements
- Ruby 3.2.3
- Rails 7

### Install

```
git clone https://github.com/timothyjamesmarias/rails_7_devise_example.git
cd rails_7_devise_example

bundle install
rails db:create
rails db:migrate

rails s
```
